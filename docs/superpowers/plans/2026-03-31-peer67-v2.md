# Peer67 v2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build an anonymous, ephemeral, encrypted messaging system for AI agents — a dead-drop relay + MCP server that lets Claude users message each other with zero metadata.

**Architecture:** A minimal Fastify relay stores encrypted blobs at hashed addresses with 24h TTL (Redis-backed). A TypeScript MCP server runs as a Claude Code subprocess, providing tools for connecting (X25519 key exchange), sending (AES-256-GCM), and receiving messages. Connection data lives locally at `~/.peer67/`.

**Tech Stack:** TypeScript, Fastify, Redis (via ioredis), @modelcontextprotocol/sdk, @noble/curves (X25519), @noble/hashes (HKDF-SHA256), @noble/ciphers (AES-256-GCM), vitest

Create `relay/src/store.ts`:

```typescript
import type { Redis } from "ioredis";
import { nanoid } from "nanoid";

const MAX_BLOB_SIZE = 65536; // 64KB
const MAX_BLOBS_PER_MAILBOX = 1000;
const TTL_SECONDS = 86400; // 24 hours
const MAILBOX_ID_REGEX = /^[0-9a-f]{64}$/;

interface Blob {
  id: string;
  blob: string;
  ts: number;
}

interface PutResult {
  id: string;
  expires_at: string;
}

export class BlobStore {
  constructor(private readonly redis: Redis) {}

  async put(mailboxId: string, blob: string): Promise<PutResult> {
    if (!MAILBOX_ID_REGEX.test(mailboxId)) {
      throw new Error("Invalid mailbox ID");
    }

    const rawSize = Buffer.byteLength(blob, "base64");
    if (rawSize > MAX_BLOB_SIZE) {
      throw new Error("Blob too large");
    }

    const key = `mb:${mailboxId}`;
    const count = await this.redis.llen(key);
    if (count >= MAX_BLOBS_PER_MAILBOX) {
      throw new Error("Mailbox full");
    }

    const id = nanoid();
    const ts = Math.floor(Date.now() / 1000);
    const entry: Blob = { id, blob, ts };

    await this.redis.rpush(key, JSON.stringify(entry));
    // Set TTL on the mailbox key. Each new blob refreshes the TTL.
    // Individual blob TTL is handled by the 24h window — the oldest
    // blob in a mailbox is at most 24h old because the key expires.
    await this.redis.expire(key, TTL_SECONDS);

    const expiresAt = new Date((ts + TTL_SECONDS) * 1000).toISOString();
    return { id, expires_at: expiresAt };
  }

  async get(mailboxId: string, after?: number): Promise<Blob[]> {
    const key = `mb:${mailboxId}`;
    const raw = await this.redis.lrange(key, 0, -1);

    const blobs: Blob[] = raw.map((r) => JSON.parse(r) as Blob);

    if (after !== undefined) {
      return blobs.filter((b) => b.ts >= after);
    }

    return blobs;
  }

  async del(mailboxId: string, blobId: string): Promise<boolean> {
    const key = `mb:${mailboxId}`;
    const raw = await this.redis.lrange(key, 0, -1);

    for (const entry of raw) {
      const parsed = JSON.parse(entry) as Blob;
      if (parsed.id === blobId) {
        const removed = await this.redis.lrem(key, 1, entry);
        return removed > 0;
      }
    }

    return false;
  }
}
```

```typescript
import Fastify from "fastify";
import cors from "@fastify/cors";
import rateLimit from "@fastify/rate-limit";
import Redis from "ioredis";
import { BlobStore } from "./store.js";
import { isValidMailboxId, isValidBase64 } from "./middleware.js";

// Use ioredis-mock in test, real Redis otherwise
async function createRedis(url: string): Promise<any> {
  if (url === "redis://mock") {
    const { default: RedisMock } = await import("ioredis-mock");
    return new RedisMock();
  }
  return new Redis(url);
}

export async function buildApp(opts?: { redisUrl?: string }) {
  const app = Fastify({
    logger: process.env.NODE_ENV !== "test"
      ? { level: process.env.LOG_LEVEL || "info" }
      : false,
  });

  await app.register(cors, { origin: true });

  if (process.env.NODE_ENV !== "test") {
    await app.register(rateLimit, { max: 60, timeWindow: "1 minute" });
  }

  const redisUrl = opts?.redisUrl || process.env.REDIS_URL || "redis://localhost:6379";
  const redis = await createRedis(redisUrl);
  const store = new BlobStore(redis);

  // PUT /d/:mailboxId — store a blob
  app.put<{
    Params: { mailboxId: string };
    Body: { blob: string };
  }>("/d/:mailboxId", async (request, reply) => {
    const { mailboxId } = request.params;
    const { blob } = request.body || {};

    if (!isValidMailboxId(mailboxId)) {
      return reply.code(400).send({ error: "Invalid mailbox ID" });
    }
    if (!blob || typeof blob !== "string" || !isValidBase64(blob)) {
      return reply.code(400).send({ error: "Missing or invalid blob" });
    }

    try {
      const result = await store.put(mailboxId, blob);
      return result;
    } catch (err: unknown) {
      const message = err instanceof Error ? err.message : "Unknown error";
      if (message === "Blob too large" || message === "Mailbox full") {
        return reply.code(400).send({ error: message });
      }
      throw err;
    }
  });

  // GET /d/:mailboxId — fetch blobs
  app.get<{
    Params: { mailboxId: string };
    Querystring: { after?: string };
  }>("/d/:mailboxId", async (request, reply) => {
    const { mailboxId } = request.params;

    if (!isValidMailboxId(mailboxId)) {
      return reply.code(400).send({ error: "Invalid mailbox ID" });
    }

    const after = request.query.after
      ? parseInt(request.query.after, 10)
      : undefined;

    const blobs = await store.get(mailboxId, after);
    return { blobs };
  });

  // DELETE /d/:mailboxId/:blobId — delete a blob
  app.delete<{
    Params: { mailboxId: string; blobId: string };
  }>("/d/:mailboxId/:blobId", async (request, reply) => {
    const { mailboxId, blobId } = request.params;

    const deleted = await store.del(mailboxId, blobId);
    if (!deleted) {
      return reply.code(404).send({ error: "Blob not found" });
    }
    return { ok: true };
  });

  // GET /health
  app.get("/health", async () => ({
    ok: true,
    service: "peer67-relay",
    version: "0.1.0",
    timestamp: new Date().toISOString(),
  }));

  return app;
}

// Start server when run directly (not imported for tests)
const isDirectRun = process.argv[1]?.endsWith("index.ts") ||
                    process.argv[1]?.endsWith("index.js");

if (isDirectRun) {
  const app = await buildApp();
  const port = parseInt(process.env.PORT || "3967", 10);
  await app.listen({ port, host: "0.0.0.0" });
  console.log(`Peer67 relay listening on :${port}`);
}

import { x25519 } from "@noble/curves/ed25519";
import { hkdf } from "@noble/hashes/hkdf";
import { sha256 } from "@noble/hashes/sha256";
import { gcm } from "@noble/ciphers/aes";
import { randomBytes } from "@noble/ciphers/webcrypto";

export interface Keypair {
  privateKey: Uint8Array;
  publicKey: Uint8Array;
}

export function generateKeypair(): Keypair {
  const privateKey = x25519.utils.randomPrivateKey();
  const publicKey = x25519.getPublicKey(privateKey);
  return { privateKey, publicKey };
}

export function deriveSharedSecret(
  myPrivateKey: Uint8Array,
  theirPublicKey: Uint8Array
): Uint8Array {
  return x25519.getSharedSecret(myPrivateKey, theirPublicKey);
}

export function deriveMailboxIds(sharedSecret: Uint8Array): {
  mailboxAtoB: Uint8Array;
  mailboxBtoA: Uint8Array;
} {
  const mailboxAtoB = hkdf(sha256, sharedSecret, undefined, "peer67-mailbox-a2b", 32);
  const mailboxBtoA = hkdf(sha256, sharedSecret, undefined, "peer67-mailbox-b2a", 32);
  return { mailboxAtoB, mailboxBtoA };
}

export function deriveEncryptKey(sharedSecret: Uint8Array): Uint8Array {
  return hkdf(sha256, sharedSecret, undefined, "peer67-encrypt", 32);
}

export function encrypt(
  key: Uint8Array,
  plaintext: string,
  mailboxIdHex: string
): string {
  const nonce = randomBytes(12);
  const aad = new TextEncoder().encode(mailboxIdHex);
  const data = new TextEncoder().encode(plaintext);

  const aes = gcm(key, nonce, aad);
  const ciphertext = aes.encrypt(data);

  // Wire format: nonce (12) + ciphertext (N + 16 auth tag)
  const wire = new Uint8Array(12 + ciphertext.length);
  wire.set(nonce, 0);
  wire.set(ciphertext, 12);

  return Buffer.from(wire).toString("base64");
}

export function decrypt(
  key: Uint8Array,
  ciphertextB64: string,
  mailboxIdHex: string
): string {
  const wire = Buffer.from(ciphertextB64, "base64");
  const nonce = wire.subarray(0, 12);
  const ciphertext = wire.subarray(12);
  const aad = new TextEncoder().encode(mailboxIdHex);

  const aes = gcm(key, nonce, aad);
  const plaintext = aes.decrypt(ciphertext);

  return new TextDecoder().decode(plaintext);
}

```typescript
import { randomBytes } from "@noble/ciphers/webcrypto";
import basex from "base-x";

const BASE62 = basex("0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz");

const PREFIX = "p67_";

export interface DecodedConnectionCode {
  publicKey: Uint8Array;
  relayUrl: string;
  nonce: Uint8Array;
}

/**
 * Encode a connection code.
 *
 * Wire format:
 *   [32 bytes] X25519 public key
 *   [2 bytes]  relay URL length (big-endian uint16)
 *   [N bytes]  relay URL (UTF-8)
 *   [16 bytes] random nonce
 */
export function encodeConnectionCode(
  publicKey: Uint8Array,
  relayUrl: string
): string {
  const urlBytes = new TextEncoder().encode(relayUrl);
  if (urlBytes.length > 65535) {
    throw new Error("Relay URL too long");
  }

  const buf = new Uint8Array(32 + 2 + urlBytes.length + 16);
  buf.set(publicKey, 0);
  buf[32] = (urlBytes.length >> 8) & 0xff;
  buf[33] = urlBytes.length & 0xff;
  buf.set(urlBytes, 34);

  const nonce = randomBytes(16);
  buf.set(nonce, 34 + urlBytes.length);

  return PREFIX + BASE62.encode(buf);
}

export function decodeConnectionCode(code: string): DecodedConnectionCode {
  if (!code.startsWith(PREFIX)) {
    throw new Error("Invalid connection code");
  }

  const encoded = code.slice(PREFIX.length);
  let buf: Uint8Array;
  try {
    buf = BASE62.decode(encoded);
  } catch {
    throw new Error("Invalid connection code");
  }

  if (buf.length < 32 + 2 + 16) {
    throw new Error("Invalid connection code");
  }

  const publicKey = buf.slice(0, 32);
  const urlLen = (buf[32] << 8) | buf[33];

  if (buf.length < 34 + urlLen + 16) {
    throw new Error("Invalid connection code");
  }

  const relayUrl = new TextDecoder().decode(buf.slice(34, 34 + urlLen));
  const nonce = buf.slice(34 + urlLen, 34 + urlLen + 16);

  return { publicKey, relayUrl, nonce };
}
```

import { createHash, randomBytes, createCipheriv, createDecipheriv } from "node:crypto";
import { hostname, userInfo } from "node:os";

interface ConnectionData {
  display_name: string;
  mailbox_send: string;  // hex, the mailbox we write to
  mailbox_recv: string;  // hex, the mailbox we read from
  shared_key: string;    // hex
  relay_url: string;
  created_at?: string;
  last_seen?: string;
}

interface PendingConnection {
  name: string;
  connection_code: string;
  private_key: string;  // hex
  public_key: string;   // hex
  created_at?: string;
}

export interface StoreData {
  version: number;
  identity: {
    name: string;
    created_at: string;
  };
  connections: Record<string, ConnectionData>;
  pending?: PendingConnection;
}

interface ContactInfo {
  name: string;
  display_name: string;
  relay_url: string;
  connected_at: string;
  last_seen: string;
}

function deriveMachineKey(): Buffer {
  const fingerprint = `${hostname()}:${userInfo().username}:peer67`;
  return createHash("sha256").update(fingerprint).digest();
}

function encryptData(data: string, key: Buffer): string {
  const iv = randomBytes(12);
  const cipher = createCipheriv("aes-256-gcm", key, iv);
  const encrypted = Buffer.concat([cipher.update(data, "utf8"), cipher.final()]);
  const tag = cipher.getAuthTag();
  // iv (12) + tag (16) + ciphertext
  return Buffer.concat([iv, tag, encrypted]).toString("base64");
}

function decryptData(encoded: string, key: Buffer): string {
  const buf = Buffer.from(encoded, "base64");
  const iv = buf.subarray(0, 12);
  const tag = buf.subarray(12, 28);
  const ciphertext = buf.subarray(28);
  const decipher = createDecipheriv("aes-256-gcm", key, iv);
  decipher.setAuthTag(tag);
  return decipher.update(ciphertext) + decipher.final("utf8");
}

export class LocalStore {
  private readonly storePath: string;
  private readonly configPath: string;
  private readonly encryptionKey: Buffer;

  constructor(dir: string) {
    this.storePath = join(dir, "store.json");
    this.configPath = join(dir, "config.json");
    this.encryptionKey = deriveMachineKey();
  }

  async init(name: string): Promise<void> {
    const dir = join(this.storePath, "..");
    if (!existsSync(dir)) {
      await mkdir(dir, { recursive: true });
    }

    if (!existsSync(this.storePath)) {
      const data: StoreData = {
        version: 1,
        identity: { name, created_at: new Date().toISOString() },
        connections: {},
      };
      await this.save(data);
    }

    if (!existsSync(this.configPath)) {
      await writeFile(
        this.configPath,
        JSON.stringify(
          {
            default_relay: "https://relay.peer67.com",
            poll_interval_seconds: 10,
          },
          null,
          2
        )
      );
    }
  }

  async load(): Promise<StoreData> {
    const encrypted = await readFile(this.storePath, "utf8");
    const json = decryptData(encrypted, this.encryptionKey);
    return JSON.parse(json) as StoreData;
  }

  private async save(data: StoreData): Promise<void> {
    const json = JSON.stringify(data, null, 2);
    const encrypted = encryptData(json, this.encryptionKey);
    await writeFile(this.storePath, encrypted);
  }

  async addConnection(name: string, conn: ConnectionData): Promise<void> {
    const data = await this.load();
    data.connections[name.toLowerCase()] = {
      ...conn,
      created_at: conn.created_at || new Date().toISOString(),
      last_seen: conn.last_seen || new Date().toISOString(),
    };
    await this.save(data);
  }

  async removeConnection(name: string): Promise<void> {
    const data = await this.load();
    delete data.connections[name.toLowerCase()];
    await this.save(data);
  }

  async listConnections(): Promise<ContactInfo[]> {
    const data = await this.load();
    return Object.entries(data.connections).map(([name, conn]) => ({
      name,
      display_name: conn.display_name,
      relay_url: conn.relay_url,
      connected_at: conn.created_at || "",
      last_seen: conn.last_seen || "",
    }));
  }

  async getConnection(name: string): Promise<ConnectionData | undefined> {
    const data = await this.load();
    return data.connections[name.toLowerCase()];
  }

  async setPending(pending: PendingConnection): Promise<void> {
    const data = await this.load();
    data.pending = {
      ...pending,
      created_at: pending.created_at || new Date().toISOString(),
    };
    await this.save(data);
  }

  async clearPending(): Promise<void> {
    const data = await this.load();
    delete data.pending;
    await this.save(data);
  }

  async getConfig(): Promise<{ default_relay: string; poll_interval_seconds: number }> {
    const raw = await readFile(this.configPath, "utf8");
    return JSON.parse(raw);
  }
}

    }

    const body = await res.json() as { blobs: RelayBlob[] };
    return body.blobs;
  }

    const res = await fetch(`${this.baseUrl}/d/${mailboxId}/${blobId}`, {
      method: "DELETE",
    });

      throw new Error(`Relay DELETE failed: ${res.status}`);
    }
  }

import { encodeConnectionCode, decodeConnectionCode } from "../codec.js";
import { LocalStore } from "../store.js";
import { RelayClient } from "../relay-client.js";

interface CreateResult {
  code: string;
  message: string;
}

interface AcceptResult {
  connected: boolean;
  name: string;
  message: string;
}

export async function connectCreate(
  store: LocalStore,
  name: string,
  relayUrl?: string
): Promise<CreateResult> {
  const config = await store.getConfig();
  const relay = relayUrl || config.default_relay;

  const kp = generateKeypair();
  const code = encodeConnectionCode(kp.publicKey, relay);

  await store.setPending({
    name,
    connection_code: code,
    private_key: Buffer.from(kp.privateKey).toString("hex"),
    public_key: Buffer.from(kp.publicKey).toString("hex"),
  });

  return {
    code,
    message: `Share this code with ${name}. Once they enter it, you'll be connected.`,
  };
}

export async function connectAccept(
  store: LocalStore,
  name: string,
  code: string
): Promise<AcceptResult> {
  const decoded = decodeConnectionCode(code);
  const theirPublicKey = decoded.publicKey;
  const relayUrl = decoded.relayUrl;

  // Generate our keypair and derive shared secret
  const kp = generateKeypair();
  const sharedSecret = deriveSharedSecret(kp.privateKey, theirPublicKey);
  const { mailboxAtoB, mailboxBtoA } = deriveMailboxIds(sharedSecret);
  const encryptKey = deriveEncryptKey(sharedSecret);

  // We are the acceptor (B).
  // We read from mailbox_a_to_b (initiator writes here).
  // We write to mailbox_b_to_a (initiator reads here).
  const mailboxSend = Buffer.from(mailboxBtoA).toString("hex");
  const mailboxRecv = Buffer.from(mailboxAtoB).toString("hex");

  // Store connection locally
  await store.addConnection(name, {
    display_name: name,
    mailbox_send: mailboxSend,
    mailbox_recv: mailboxRecv,
    shared_key: Buffer.from(encryptKey).toString("hex"),
    relay_url: relayUrl,
  });

  // Send our public key to the initiator's recv mailbox so they can complete
  const client = new RelayClient(relayUrl);
  const handshakeBlob = Buffer.from(kp.publicKey).toString("base64");
  await client.put(mailboxRecv, handshakeBlob);

  return {
    connected: true,
    name,
    message: `Connected with ${name}. You can now send messages.`,
  };
}

/**
 * Check if a pending outgoing connection has been completed.
 * Called during polling. If the acceptor's public key is in our mailbox,
 * we complete the handshake.
 */
export async function connectComplete(store: LocalStore): Promise<string | null> {
  const data = await store.load();
  if (!data.pending) return null;

  const pending = data.pending;
  const privateKey = Uint8Array.from(Buffer.from(pending.private_key, "hex"));
  const publicKey = Uint8Array.from(Buffer.from(pending.public_key, "hex"));

  // We need to figure out which mailbox to check.
  // As initiator, the acceptor will write their public key to mailbox_b_to_a.
  // But we don't know the mailbox IDs yet — we need the acceptor's public key first.
  // So we use a temporary "handshake mailbox" derived from our public key alone.
  // Wait — the acceptor sends their public key to mailbox_a_to_b (recv from initiator's perspective).
  // But we haven't derived mailbox IDs yet either, because we don't have the shared secret.
  //
  // Solution: use a "rendezvous mailbox" = SHA256(initiator_public_key) as hex.
  // The acceptor also computes this from the connection code and writes there.
  const { createHash } = await import("node:crypto");
  const rendezvousMailbox = createHash("sha256")
    .update(publicKey)
    .digest("hex");

  const config = await store.getConfig();
  // The relay URL was encoded in the connection code. Parse it.
  const { decodeConnectionCode } = await import("../codec.js");
  const decoded = decodeConnectionCode(pending.connection_code);
  const relayUrl = decoded.relayUrl;

  const client = new RelayClient(relayUrl);
  const blobs = await client.get(rendezvousMailbox);

  if (blobs.length === 0) return null;

  // The first blob should be the acceptor's public key
  const theirPublicKey = Uint8Array.from(Buffer.from(blobs[0].blob, "base64"));
  if (theirPublicKey.length !== 32) return null;

  // Derive shared secret and keys
  const sharedSecret = deriveSharedSecret(privateKey, theirPublicKey);
  const { mailboxAtoB, mailboxBtoA } = deriveMailboxIds(sharedSecret);
  const encryptKey = deriveEncryptKey(sharedSecret);

  // We are the initiator (A).
  // We write to mailbox_a_to_b.
  // We read from mailbox_b_to_a.
  const mailboxSend = Buffer.from(mailboxAtoB).toString("hex");
  const mailboxRecv = Buffer.from(mailboxBtoA).toString("hex");

  await store.addConnection(pending.name, {
    display_name: pending.name,
    mailbox_send: mailboxSend,
    mailbox_recv: mailboxRecv,
    shared_key: Buffer.from(encryptKey).toString("hex"),
    relay_url: relayUrl,
  });

  // Clean up: delete handshake blob, clear pending
  await client.del(rendezvousMailbox, blobs[0].id);
  await store.clearPending();

  return pending.name;
}

import { encrypt } from "../crypto.js";
import { LocalStore } from "../store.js";
import { RelayClient } from "../relay-client.js";

interface SendResult {
  sent: boolean;
  expires_at: string;
}

export async function sendMessage(
  store: LocalStore,
  to: string,
  message: string
): Promise<SendResult> {
  const conn = await store.getConnection(to);
  if (!conn) {
    throw new Error(`No connection with "${to}". Connect first.`);
  }

  const key = Uint8Array.from(Buffer.from(conn.shared_key, "hex"));
  const mailbox = conn.mailbox_send;

  // Build envelope
  const envelope = JSON.stringify({
    v: 1,
    t: Math.floor(Date.now() / 1000),
    b: message,
  });

  // Encrypt with AAD = mailbox ID
  const ciphertext = encrypt(key, envelope, mailbox);

  // Send to relay
  const client = new RelayClient(conn.relay_url);
  const result = await client.put(mailbox, ciphertext);

  return { sent: true, expires_at: result.expires_at };
}

```typescript
import { decrypt } from "../crypto.js";
import { LocalStore } from "../store.js";
import { RelayClient } from "../relay-client.js";

interface InboxMessage {
  from: string;
  body: string;
  timestamp: string;
  blob_id: string;
  relay_url: string;
  mailbox: string;
}

export async function checkInbox(
  store: LocalStore,
  from?: string
): Promise<InboxMessage[]> {
  const data = await store.load();
  const messages: InboxMessage[] = [];

  const connectionNames = from
    ? [from.toLowerCase()]
    : Object.keys(data.connections);

  for (const name of connectionNames) {
    const conn = data.connections[name];
    if (!conn) continue;

    const client = new RelayClient(conn.relay_url);
    const key = Uint8Array.from(Buffer.from(conn.shared_key, "hex"));
    const mailbox = conn.mailbox_recv;

    try {
      const blobs = await client.get(mailbox);

      for (const blob of blobs) {
        try {
          const envelopeJson = decrypt(key, blob.blob, mailbox);
          const envelope = JSON.parse(envelopeJson) as {
            v: number;
            t: number;
            b: string;
          };

          messages.push({
            from: conn.display_name,
            body: envelope.b,
            timestamp: new Date(envelope.t * 1000).toISOString(),
            blob_id: blob.id,
            relay_url: conn.relay_url,
            mailbox,
          });
        } catch {
          // Skip blobs we can't decrypt (could be handshake blobs or corrupted)
        }
      }
    } catch {
      // Skip unreachable relays
    }
  }

  // Sort newest first
  messages.sort((a, b) => b.timestamp.localeCompare(a.timestamp));
  return messages;
}

export async function acknowledgeMessages(
  messages: InboxMessage[]
): Promise<void> {
  for (const msg of messages) {
    const client = new RelayClient(msg.relay_url);
    await client.del(msg.mailbox, msg.blob_id);
  }
}

Anonymous, encrypted, ephemeral messaging for AI agents.

Two humans communicate through their Claude sessions. Messages are end-to-end encrypted — the relay server stores only opaque blobs at hashed addresses. It never knows who is talking, what they're saying, or who the messages are for. Everything auto-deletes after 24 hours.

## Architecture

```
Human ↔ Claude ↔ MCP Server ↔ Relay ↔ MCP Server ↔ Claude ↔ Human
                  (encrypt)    (blob)   (decrypt)
```

- **Relay** — a dumb key-value store. Holds encrypted blobs at hashed addresses. No users, no auth, no logs. 24h TTL.
- **MCP Server** — runs as a Claude Code subprocess. Handles key exchange (X25519), encryption (AES-256-GCM), and local connection storage.
- **Connection codes** — encode an X25519 public key + relay URL. Share it however you want. One-time use.
- **Two mailboxes per connection** — each direction gets its own address. The relay can't even tell they're related.

## Self-host a relay

```bash
docker compose up -d
```

Or deploy the `relay/` directory anywhere that runs Node.js + Redis. Set `REDIS_URL` and you're done.

The default relay is `relay.peer67.com`. Override per-connection or in `~/.peer67/config.json`.

## Security

- **Zero-knowledge relay** — stores only encrypted blobs at hashed addresses
- **No metadata** — no users, no accounts, no sender/recipient fields, no logs
- **End-to-end encryption** — AES-256-GCM with keys derived via X25519 + HKDF
- **Ephemeral** — 24h auto-delete, no permanent storage
- **No correlation** — separate mailbox IDs per direction prevent linking sender and recipient
- **AAD binding** — ciphertext is bound to its destination mailbox, preventing replay attacks

```markdown
# Peer67 Protocol v1

## Overview

Peer67 is an anonymous, ephemeral messaging protocol. Two parties establish a connection via a one-time code exchange, then send encrypted messages through a relay server that has zero knowledge of the participants.

## Relay API

A Peer67 relay exposes three endpoints:

| Method | Path | Description |
|--------|------|-------------|
| `PUT /d/{mailbox_id}` | Store a blob. Body: `{ "blob": "<base64>" }`. Returns `{ "id": "<nanoid>", "expires_at": "<ISO>" }`. |
| `GET /d/{mailbox_id}?after={unix_ts}` | Fetch blobs. Returns `{ "blobs": [{ "id", "blob", "ts" }] }`. |
| `DELETE /d/{mailbox_id}/{blob_id}` | Delete a blob. Returns `{ "ok": true }`. |

Constraints:
- `mailbox_id`: 64-character lowercase hex string (256-bit)
- Max blob size: 64 KB (base64-encoded)
- Max blobs per mailbox: 1000
- All blobs auto-expire after 86400 seconds (24 hours)
- No authentication required

## Connection Protocol

### Step 1: Initiator creates a code

1. Generate X25519 keypair `(sk_a, pk_a)`
2. Encode connection code: `p67_` + base62 of `pk_a (32B) || url_len (2B big-endian) || relay_url (NB UTF-8) || nonce (16B random)`
3. Share code out-of-band

### Step 2: Acceptor processes the code

1. Decode `pk_a`, `relay_url` from code
2. Generate X25519 keypair `(sk_b, pk_b)`
3. Compute `shared = X25519(sk_b, pk_a)`
4. Derive via HKDF-SHA256 (no salt):
   - `mailbox_a2b = HKDF(shared, info="peer67-mailbox-a2b", len=32)`
   - `mailbox_b2a = HKDF(shared, info="peer67-mailbox-b2a", len=32)`
   - `encrypt_key = HKDF(shared, info="peer67-encrypt", len=32)`
5. Store connection locally
6. Compute rendezvous mailbox: `SHA256(pk_a)` as hex
7. PUT `pk_b` (base64) to rendezvous mailbox on relay

### Step 3: Initiator completes

1. Compute rendezvous mailbox: `SHA256(pk_a)` as hex
2. Poll relay for blobs at rendezvous mailbox
3. When found: decode `pk_b`, compute `shared = X25519(sk_a, pk_b)`
4. Derive same `mailbox_a2b`, `mailbox_b2a`, `encrypt_key`
5. Store connection locally
6. DELETE the handshake blob from rendezvous mailbox

### Roles

- **Initiator (A)**: writes to `mailbox_a2b`, reads from `mailbox_b2a`
- **Acceptor (B)**: writes to `mailbox_b2a`, reads from `mailbox_a2b`

## Message Format

Plaintext envelope:
```json
{ "v": 1, "t": <unix_seconds>, "b": "<message body>" }
```

Encryption:
- Algorithm: AES-256-GCM
- Key: `encrypt_key` (32 bytes from HKDF)
- Nonce: 12 random bytes
- AAD: mailbox_id (hex string, UTF-8 encoded)

Wire format (base64 of):
```
[12B nonce] [NB ciphertext + 16B GCM auth tag]
```

The blob sent to the relay is the base64 encoding of this wire format.

## Security Properties

- The relay never learns participant identities
- Separate mailbox IDs per direction prevent correlation
- AAD binding prevents cross-mailbox replay
- 24h TTL ensures forward secrecy of metadata
- X25519 provides authenticated key exchange (assuming trusted code delivery)
