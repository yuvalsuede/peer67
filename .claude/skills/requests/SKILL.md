---
name: peer67:requests
description: View, accept, or decline incoming connection requests
---

Use ONLY the `peer67_requests` MCP tool. Do NOT read files or code.

Call `peer67_requests` with action="list" to show pending requests.

If user says "accept X", call with action="accept" and from="X".
If user says "decline X", call with action="decline" and from="X".
